# Issue Tracker

Issues are tracked on **GitHub Issues** for the repository `dylanpan/MergeR`.

## Commands

- `gh issue create` — create a new issue
- `gh issue view <number>` — view issue details
- `gh issue list` — list open issues
- `gh issue close <number>` — close an issue
- `gh issue comment <number> --body "<text>"` — add a comment

## Workflow

1. New issues are created without labels (raw)
2. Triage assigns labels and moves through the state machine
3. Fully specified issues → `ready-for-agent` for AFK execution
